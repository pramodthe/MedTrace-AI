# Clinical Intelligence Design System

## Visual Identity
A clean, high-fidelity medical documentation interface designed for clarity, trust, and professional efficiency. The system prioritizes readability and structured data visualization.

## Typography
- **Primary Font:** Inter (Sans-serif)
- **Heading 1:** 24px, Semi-bold, #0f172a (Primary Text)
- **Heading 2:** 20px, Semi-bold, #1e293b
- **Body Text:** 14px, Regular, #334155
- **Labels/Metadata:** 12px, Medium, #64748b

## Color Palette

### Core Surfaces
- **Surface:** `#f7f9fb` (Page background)
- **Surface-Container:** `#ffffff` (Main cards and panels)
- **Border:** `#e2e8f0` (Subtle dividers and card outlines)

### Primary & Interaction
- **Primary:** `#0052cc` (Primary actions, active states)
- **Primary-Container:** `#e0e7ff` (Selected message bubbles)
- **Interactive-Hover:** `#f1f5f9` (Button/List item hover)

### Semantic States
- **Success:** `#22c55e` (Completed states, high match percentage)
- **Warning/Alert:** `#facc15` (Important notes, illustrative purposes disclaimer)
- **Notification/Important:** `#3b82f6` (Active processing, "Refining..." states)
- **Error:** `#ef4444` (Critical contradictions or data failures)

## UI Components

### Cards & Containers
- **Border Radius:** 12px (Standard containers), 24px (Message bubbles/Floating bars)
- **Shadow:** Low-elevation soft shadows (`0 1px 3px 0 rgb(0 0 0 / 0.1)`)
- **Padding:** 24px for main panels, 16px for internal cards

### Transcription & Messaging
- **Clinician Bubble:** Light blue background (`#eff6ff`), left-aligned.
- **Patient Bubble:** Light grey background (`#f1f5f9`), right-aligned with avatar.
- **AI Command Bar:** Pill-shaped, floating, centrally aligned at the bottom with a 50% blur backdrop.

### Layout Rhythm
- **Column Gap:** 24px
- **Vertical Spacing:** 16px between internal elements
- **Section Headers:** 32px top margin to distinguish primary regions (Transcript vs. Insights)

## Interaction Principles
- **Live Status:** Use subtle pulse animations or color-coded chips (e.g., "98% Match") for real-time AI feedback.
- **Accessibility:** Ensure a minimum contrast ratio of 4.5:1 for all medical data.
- **Contextual Actions:** Floating inputs should always remain accessible without obstructing primary documentation.
